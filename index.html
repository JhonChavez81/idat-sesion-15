Fundamentos de CSS

El arte de dar estilo a la web

Usa las flechas del teclado o los botones para navegar

¿Qué es CSS?

  • Cascading Style Sheets (Hojas de Estilo en Cascada).
  • Se encarga de la presentación visual: colores, tipografía, espacios y maquetación.
  • Separa el contenido (HTML) del diseño visual.
  • Permite el diseño Responsive (adaptable a móviles y escritorio).

3 Formas de Insertar CSS

En Línea (Inline)

Se usa el atributo style en la etiqueta HTML.

<h1 style="color:red;">
Texto
</h1>

Interno

Se usa la etiqueta <style> en el <head>.

<style>
  h1 { color: blue; }
</style>

Externo ⭐ MEJOR

Se vincula un archivo .css independiente.

<link rel="stylesheet"
href="style.css">

Selectores Básicos

Indican a qué elementos HTML se les aplicarán los estilos.

Símbolo Nombre Aplicación Ejemplo
* Universal Absolutamente todo * { margin: 0; }
p, div Etiqueta Todas las etiquetas de ese tipo p { color: white; }
. (punto) Clase Elementos con class="nombre" .caja { padding: 10px; }
# (gato) ID Elemento único con id="nombre" #menu { background: red; }

Cascada y Especificidad

La Cascada

CSS se lee de arriba hacia abajo. Si hay reglas en conflicto, la última escrita gana.

Especificidad

Es el "peso" o prioridad de un selector.

Jerarquía de peso:

  1. Estilos en línea style="" (Más fuerte)
  2. IDs #header
  3. Clases .btn
  4. Etiquetas h1 (Más débil)
